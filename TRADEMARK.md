# Trademark Policy

**"NeuronEdge Enclave"**, **"NeuronEdge"**, the NeuronEdge Enclave wordmark,
and the NeuronEdge Enclave logo are trademarks of **Infrastacks LLC** (the
"NeuronEdge Enclave Trademarks").

This policy explains how you may — and may not — use the NeuronEdge Enclave
Trademarks. It exists to prevent confusion in the market and to protect the
community's ability to tell Infrastacks's project apart from forks and
derivative offerings. This is the same model Mozilla applies to Firefox.

## The code is open; the brand is not

The NeuronEdge Enclave source code is licensed under the Apache License 2.0,
which permits nearly any use — including commercial use, modification, and
redistribution. **A license to the code is not a license to the
NeuronEdge Enclave Trademarks.** Using the trademarks requires separate
permission under this policy.

## Permitted uses (no permission needed)

You may use the NeuronEdge Enclave Trademarks without asking, for:

- **Referencing the project.** "NeuronEdge Enclave is an open-source agent
  runtime …", linking to this repository, describing compatibility.
- **Forks and derivatives that clearly disambiguate.** A fork must be named
  and branded distinctly (e.g., "AcmeSandbox, powered by NeuronEdge Enclave")
  so no one could reasonably confuse it with the Infrastacks project. The
  fork's own name must come first.
- **Truthful, non-misleading attribution.** "Built on NeuronEdge Enclave" or
  "NeuronEdge Enclave-compatible" when accurate.

## Uses that require written permission

You **must** contact Infrastacks (eng@infrastacks.com) before:

- **Distributing a product named "NeuronEdge Enclave"** or any name so similar
  as to cause likely confusion.
- **Using the NeuronEdge Enclave logo** in a product, service, website, or
  marketing material (the wordmark policy above is more permissive; the logo
  is not).
- **Offering a commercial service** that uses the NeuronEdge Enclave
  Trademarks in its name (e.g., a hosted or managed service).

Infrastacks will not unreasonably withhold permission for uses that are
truthful, non-confusing, and consistent with open-source norms.

## Prohibited uses

You may **not**:

- Use the NeuronEdge Enclave Trademarks in a way that implies Infrastacks
  endorses or is affiliated with your product when it is not.
- Use a name or logo confusingly similar to suggest your fork *is* the
  Infrastacks project or an official distribution.
- Register or apply to register the NeuronEdge Enclave Trademarks (or
  confusingly similar marks) as a trademark, domain, or social handle.

## Why this matters

An open-source infrastructure project's most durable asset is often its name
and the trust attached to it. This policy lets the code remain freely usable
under Apache-2.0 while ensuring that "NeuronEdge Enclave" keeps identifying
the Infrastacks project — not a relabeled competitor. This is the same
protection that lets Mozilla ship Firefox under a permissive license while
Debian historically shipped it as "Iceweasel."
